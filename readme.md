# Simulador de Investimentos em Fundos Imobiliários (FII)

Projeto desenvolvido como parte do Bootcamp de Excel com IA da DIO, em parceria com o Santander Open Academy.

Este repositório contém uma planilha em Excel criada para simular investimentos em fundos imobiliários, permitindo ao usuário visualizar de forma prática o impacto de seus aportes ao longo do tempo.

---

##  Descrição do Projeto

A ferramenta foi desenvolvida com o objetivo de auxiliar investidores iniciantes a compreender melhor como funcionam os investimentos em Fundos Imobiliários (FII).

A planilha permite realizar simulações a partir de variáveis comuns no universo dos investimentos, como valor investido, perfil do investidor e taxa de rendimento, automatizando os principais cálculos financeiros e facilitando a tomada de decisão.

---

## Funcionalidades

- Simulação de investimentos em Fundos Imobiliários
- Distribuição do investimento conforme o perfil do investidor
- Cálculo automático do valor total investido
- Cálculo do patrimônio acumulado
- Estimativa de dividendos mensais
- Estrutura simples e acessível para usuários iniciantes

---

## Como Utilizar a Planilha

1. Abra o arquivo `Projeto FII - DIO.xlsx`
2. Informe os dados solicitados nos campos indicados
3. Selecione o perfil de investidor desejado
4. A planilha realizará automaticamente os cálculos
5. Analise os resultados de patrimônio e dividendos gerados
